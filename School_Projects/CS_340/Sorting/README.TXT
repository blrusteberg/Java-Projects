
PROJECT TITLE: Project1_Sorting

PURPOSE OF PROJECT: The purpose of this project is to implement InsertionSort, MergeSort, 
and HeapSort, including the efficient linear time BuildHeap procedure for the latter.

VERSION or DATE: 2/3/2019

AUTHOR: Blake Rusteberg

TIPS: 
1) The example files are examples of what the output files will look like. 
 Example_HeapSort_Files
 Example_InsertionSort_Files
 Example_MergeSort_Files
 

USER INSTRUCTIONS:
1) Download the Project1_Sorting.zip to the Desktop

2) Extract Project1_Sorting.zip to the Desktop

3) Open the Project1_Sorting file and and copy the path in the directory search
 | Example: C:\Users\blrus\Desktop\Project1_Sorting |
 
4) Open command prompt by typing "cmd" into the windows search bar

5) Type "cd" and paste the path you copied from earlier
 | Example: cd C:\Users\blrus\Desktop\Project1_Sorting |

6) Type "java -jar Project1_Sorting.jar" and the program will be executed

7) Follow the instructions in the program to generate output files for 
the given sorting algorithms

8) To re-run the program go to step 6.
