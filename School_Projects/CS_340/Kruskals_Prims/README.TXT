
PROJECT TITLE: Program_4_Kruskals_Prims

PURPOSE OF PROJECT: The purpose of this project is to implement Kruskal's and Prim's Algorithm 
for finding MSTs of undirected positively weighted graphs.

VERSION or DATE: 4/2/2019

AUTHOR: Blake Rusteberg

USER INSTRUCTIONS:
1) Download the Program_4_Kruskals_Prims.zip to the Desktop

2) Extract Program_4_Kruskals_Prims.zip to the Desktop

3) Open the Program_4_Kruskals_Prims.zip file and and copy the path in the directory search
 | Example: C:\Users\blrus\Desktop\Program_4_Kruskals_Prims |
 
4) Open command prompt by typing "cmd" into the windows search bar

5) Type "cd" and paste the path you copied from earlier
 | Example: cd C:\Users\blrus\Desktop\Program_4_Kruskals_Prims |

6) Type "java -jar Program_4_Kruskals_Prims.jar" and the program will be executed

7) Follow the instructions in the program 

8) To re-run the program go to step 6.
