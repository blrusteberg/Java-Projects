
PROJECT TITLE: Program_3_Topological

PURPOSE OF PROJECT: The purpose of this project is to run Depth First Search on 3 graphs and 
detect whether they are cyclic or not.

VERSION or DATE: 3/17/2019

AUTHOR: Blake Rusteberg


USER INSTRUCTIONS:
1) Download the Program_3_Topological.zip to the Desktop

2) Extract Program_3_Topological.zip to the Desktop

3) Open the Program_3_Topological file and and copy the path in the directory search
 | Example: C:\Users\blrus\Desktop\Program3_Topological |
 
4) Open command prompt by typing "cmd" into the windows search bar

5) Type "cd" and paste the path you copied from earlier
 | Example: cd C:\Users\blrus\Desktop\Program_3_Topological |

6) Type "java -jar Program_3_Topological.jar" and the program will be executed

7) Follow the instructions in the program 

8) To re-run the program go to step 6.
